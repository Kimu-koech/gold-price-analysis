# Gold Price Project

## Overview

This project analyzes historical gold prices to understand market behavior and prepare the data for machine learning-based price prediction.

The project includes data collection, cleaning, exploratory data analysis (EDA), and will later be extended with feature engineering and predictive modeling.

## Dataset

- Source: Yahoo Finance
- Asset: Gold (XAU/USD)
- Frequency: Daily

## Objectives

- Clean historical gold price data
- Perform exploratory data analysis
- Analyze price trends
- Measure market volatility
- Create useful visualizations
- Prepare data for machine learning
